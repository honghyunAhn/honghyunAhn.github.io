Create React App を使って Typescript プロジェクトを生成する方法は簡単です。

## ① Create React App 設置

```bash
npm install -g create-react-app
```

## ② Typescript テンプレート作成

```bash
npx create-react-app プロジェクト名 --template typescript
```

